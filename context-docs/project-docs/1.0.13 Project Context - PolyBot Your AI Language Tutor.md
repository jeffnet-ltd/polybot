## Executive Summary

Polybot is an **AI-powered multilingual language learning platform** that runs entirely locally. It combines structured CEFR-based curriculum with free-flowing AI roleplay to provide a "True Bilingual" learning experience.

**Key Differentiator:** Users learn in their target language while receiving explanations and UI support in their native language. The AI runs locally (Llama 3 8B), ensuring total privacy and zero latency.

## Technical Architecture

### Stack Overview

- **Frontend:** React 18.2.0 + Tailwind CSS + Lucide React
    
- **Backend:** FastAPI (Python 3.11) + **Llama 3 8B Instruct** (via Hugging Face `transformers`)
    
- **Auth:** **Google OAuth 2.0** (via `authlib`) + Local JWT/Session Management
    
- **Database:** MongoDB 7.0
    
- **Infrastructure:** Docker + Docker Compose (V2) - Docker Desktop (WSL2 Backend)
    

### Hardware Requirement

- **AI Model:** Llama 3 8B Instruct loaded with `device_map="auto"` and `bfloat16` precision on NVIDIA CUDA GPU.

## Core Features & User Flow

1. **Registration/Login:**
    
    - Social Login (Google) or Standard Email/Password.
        
    - **Smart Onboarding:** New users (or those with 0 XP) are automatically routed to the _Language Setup_ screen to select their Native and Target languages.
        
2. **Dashboard (Main Screen):**
    
    - Displays progress (XP, Words Learned, Streak).

3. **Curriculum (CEFR A1):**
    
    - Structured lessons (Greetings, Introductions).
        
    - Interactive Exercises: Flashcards, Fill-in-the-blank, Sentence Building, Matching.
        
4. **AI Tutor (Boss Fight):**
    
    - **Scenario:** "Introduce Yourself".
        
    - **Mechanism:** AI starts the chat. User responds. AI assesses progress (Guardrail) and corrects grammar (Correction Prompt) in real-time.
        
    - **Completion:** Successfully stating name triggers "Mission Complete" and awards XP.
