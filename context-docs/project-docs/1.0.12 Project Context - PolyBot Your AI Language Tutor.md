## Executive Summary

Polybot is an **AI-powered multilingual language learning platform** that runs entirely locally. It provides structured conversational practice based on the **CEFR** curriculum.

The platform has successfully migrated to the Llama 3 model for robust multilingual chat and features stable, goal-driven conversational flows, maintaining the core value proposition of private, effective language practice.

## Core Value Proposition (Login Screen/Value Proposition Screen Context)

- **Core Differentiator:** True **bilingual learning**â€”users receive explanations in their native language while conversing solely in the target language.
    
- **Structured Curriculum:** Learning is based on the **CEFR** standard, providing clear, structured progress tracking.
    
- **Privacy & Speed:** The AI model runs **entirely locally** on the user's GPU (NVIDIA CUDA), providing near-instant response times and ensuring maximum data privacy.
    
- **Gamification:** Users can earn **XP** and progress through interactive lessons (Flashcards, Matching, Sentence Building).
    

## Technical Architecture

### Deployment Scope: Multi-Device (App Conversion)

The application is built on a **React/FastAPI stack**, making it highly adaptable:

- **Primary Target:** Web application optimized for desktop and tablet browsers (using Tailwind CSS for responsiveness).
    
- **Future Scope (App Conversion):** The React frontend is suitable for conversion to a **cross-platform desktop application** (e.g., Electron) or a **mobile/tablet application** (e.g., React Native/Capacitor bridge with local backend) to run on all major devices.
    
- **Local AI Core:** The core backend (`FastAPI` + `Llama 3`) is designed to run locally, ensuring consistent performance regardless of the host operating system (Windows/Linux/macOS).
    

### Stack Overview

```
Frontend: React 18.2.0 + Tailwind CSS + Lucide React
Backend: FastAPI (Python 3.11) + Llama 3 AI (Instruct Tuned)
Database: MongoDB 7.0
Infrastructure: Docker + Docker Compose (V2) - Docker Desktop (WSL2 Backend)
AI Model: Llama 3 8B Instruct (Loaded on NVIDIA CUDA GPU using device_map="auto") 
```
